How to run the project:
1. Open the project in xcode
2. Run the project in a iPhone
3. It will show a very simple interface with one button
4. Press create and Save Video button.
Processed image will be shown bellow the button.After finishing processing it will show a alert window saying video saved in photo gallery. (Text, Mp3 and image files are already in the project file)
5. Now open the photo gallery and you can find your desired video
6.You can select and Play the Video
